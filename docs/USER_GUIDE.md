## 一、参赛者使用指南

### 1.1 注册与登录

#### 注册账号
1. 打开 `http://localhost:3000/register`
2. 输入用户名（唯一，不可重复）
3. 输入密码（至少 6 位）
4. 确认密码
5. 点击「注册」按钮

#### 登录
1. 打开 `http://localhost:3000/login`
2. 输入用户名和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到首页

> 💡 登录后状态会保持 24 小时，关闭浏览器后再次访问无需重新登录。

---

### 1.2 首页（Dashboard）

登录后进入首页，包含以下模块：

| 模块 | 说明 |
|------|------|
| **统计卡片** | 显示正在进行的竞赛数、平均得分、当前排名 |
| **能力分布** | 雷达图展示各安全领域的掌握情况 |
| **下一场比赛** | 显示最近即将开始的竞赛 |
| **最近提交** | 显示最近的竞赛成绩记录 |
| **最新公告** | 平台公告信息 |

---

### 1.3 参加竞赛

#### 浏览竞赛列表
1. 点击顶部导航栏的「竞赛」或移动端底部「竞赛」Tab
2. 可使用筛选器查看「全部 / 进行中 / 即将开始 / 已结束」的竞赛
3. 已参加的竞赛会显示「已参加」标签和得分

#### 参赛流程
1. 在竞赛列表中点击「立即参赛」进入竞赛详情页
2. 确认竞赛信息（时间、题目数、时长等）
3. 点击「立即参赛」按钮进入答题页面

#### 答题页面

```
┌─────────────────────────────────────────────────┐
│ ← 返回    第三届网络安全知识竞赛    ⏱ 01:23:45   │
├──────────────┬──────────────────────────────────┤
│              │                                  │
│  📋 答题卡   │  第 12/50 题    [单选题]  [3分]  │
│  ┌─────────┐ │                                  │
│  │ 1  2  3 │ │  题目内容...                      │
│  │ 4  5  6 │ │                                  │

│  │10 11 12 │ │  ○ B. 选项二                      │

│  │         │ │  [上一题]  [标记此题]  [下一题]    │

└──────────────┴──────────────────────────────────┘
```

**答题卡说明**：
- 🔵 蓝色高亮 = 当前题目
- 🔵 蓝色边框 = 已答题目
- ⚪ 灰色 = 未答题目
- ⭐ 星标 = 标记的不确定题目

**答题规则**：
- ⏱ 倒计时结束自动交卷
- 🚫 离开页面（切换标签/最小化）自动交卷
- 📝 可自由切换题目，支持标记不确定的题目
- ✅ 提交前会提示未答题数

#### 四种题型操作

| 题型 | 操作方式 |
|------|---------|
| **单选题** | 点击选项即可选择，选 A 后 B 自动取消 |
| **多选题** | 点击选项勾选/取消，需选对全部才得分 |
| **判断题** | 点击「正确」或「错误」按钮 |
| **填空题** | 在输入框中输入答案，不区分大小写 |

#### 提交试卷
1. 点击「提交试卷」按钮
2. 系统弹窗提示未答题数和标记题数
3. 确认无误后点击「确认提交」
4. 提交成功后跳转到成绩页面

---

### 1.4 查看成绩

提交试卷后自动进入成绩页面，显示：

- 🎯 得分（带圆形进度图 + 数字滚动动画）
- 📊 正确率
- ⏱ 用时
- 🏆 排行榜 TOP 5
- 🏅 成就标签（如「满分大神」「闪电侠」等）

**成绩评价**：

| 分数段 | 评价 |
|--------|------|
| 90%+ | 🎉 太棒了！你是网络安全高手 |
| 80-89% | 👏 优秀！表现非常出色 |
| 70-79% | 👍 不错！基础扎实 |
| 60-69% | 💪 及格了！继续加油 |
| <60% | 📚 还需努力，多练习 |

---

### 1.5 排行榜

1. 点击顶部导航栏的「排行」
2. 显示当前进行中竞赛的排行榜
3. 前三名使用 🥇🥈🥉 标识
4. 按分数降序排列，同分按用时升序

---

### 1.6 题库练习

1. 点击顶部导航栏的「题库」
2. 可按分类和难度筛选题目
3. 点击「开始练习」进入练习模式
4. 答题后即时显示对错和解析
5. 答错的题目自动加入错题本

---

### 1.7 错题本

1. 点击顶部导航栏「我的」→「错题本」
2. 显示所有答错的题目
3. 每道题显示：
   - 你的答案（红色）
   - 正确答案（绿色）
   - 解析说明
   - 错误次数
4. 可点击「移除」删除已掌握的错题

---

### 1.8 个人中心

1. 点击顶部导航栏「我的」或右上角头像
2. 查看个人信息：用户名、注册时间、参赛次数、总排名
3. 可修改密码

## 二、管理员使用指南

### 2.1 进入后台

1. 使用管理员账号登录（默认：`admin / admin123`）
2. 访问 `http://localhost:3000/admin`

---

### 2.2 管理仪表盘

显示平台核心数据：
- 总用户数
- 总题目数
- 总竞赛数
- 总提交数

---

### 2.3 题目管理

#### 新增题目
1. 进入「题目管理」页面
2. 点击「新增题目」按钮
3. 填写题目信息：
   - **分类**：Web安全 / 密码学 / 逆向工程 / Pwn / Misc / 网络安全基础 / 操作系统安全 / 安全法规与合规
   - **难度**：简单 / 中等 / 困难
   - **题型**：单选 / 多选 / 判断 / 填空
   - **题目内容**
   - **选项**（单选/多选题需要）
   - **正确答案**
   - **解析**（可选）
   - **标签**（可选）
   - **分值**（默认 2 分）
4. 点击「保存」

#### 批量导入
1. 点击「批量导入」按钮
2. 上传 CSV 文件或粘贴 JSON 数据
3. 预览导入结果
4. 确认导入

**CSV 格式示例**：
```csv
category,difficulty,type,title,options,answer,explanation,tags,score
Web安全,easy,single,SQL注入属于什么攻击?,"[""A.注入"",""B.XSS"",""C.CSRF"",""D.中间人""]","A.注入",SQL注入是最典型的注入攻击,"[""SQL注入""]",2
```

#### 编辑/删除题目
1. 在题目列表中找到目标题目
2. 点击「编辑」修改题目信息
3. 点击「删除」软删除题目（不会真正删除，只是隐藏）

---

### 2.4 竞赛管理

#### 创建竞赛
1. 进入「竞赛管理」页面
2. 点击「创建竞赛」按钮
3. 填写竞赛信息：
   - **标题**：竞赛名称
   - **描述**：竞赛说明
   - **开始时间**：竞赛开始时间
   - **结束时间**：竞赛结束时间
   - **答题时长**：分钟数
   - **选择题目**：从题库中选择题目
   - **随机题目顺序**：是/否
   - **随机选项顺序**：是/否
4. 点击「保存」

#### 发布/编辑竞赛
1. 在竞赛列表中找到目标竞赛
2. 点击「编辑」修改竞赛信息
3. 修改状态为 `ongoing` 即可发布竞赛

#### 删除竞赛
1. 点击「删除」按钮
2. 确认后删除竞赛及其关联数据

---

### 2.5 用户管理

1. 进入「用户管理」页面
2. 查看用户列表（用户名、角色、注册时间）
3. 可修改用户角色（user ↔ admin）
4. 可禁用用户账号

## 三、常见问题 FAQ

### Q1: 忘记密码怎么办？
A: 请联系管理员重置密码。当前版本暂不支持自助找回密码。

### Q2: 答题时不小心离开页面怎么办？
A: 离开页面会自动交卷，答案按离开时的状态提交。无法恢复继续答题。

### Q3: 多选题少选了一个选项能得分吗？
A: 不能。多选题必须全对才得分，多选、漏选、错选均不得分。

### Q4: 填空题答案大小写有影响吗？
A: 没有影响。系统会自动忽略大小写和首尾空格。

### Q5: 竞赛结束后还能查看成绩吗？
A: 可以。在竞赛列表中点击已参加的竞赛，可查看历史成绩。

### Q6: 如何查看自己的薄弱知识点？
A: 在首页的「能力分布」雷达图中，可以看到各安全领域的掌握情况。同时错题本中也按分类整理了答错的题目。

### Q7: 管理员如何快速建题库？
A: 使用「批量导入」功能，准备 CSV 格式的题目数据，一次性导入多道题目。

### Q8: 竞赛可以设置团队赛吗？
A: 当前版本仅支持个人赛，团队赛功能在后续版本中规划。

### Q9: 答题时网络断了怎么办？
A: 如果网络恢复后仍在竞赛时间内，可以重新进入答题页面继续。如果已超时或离开页面，系统会自动交卷。

### Q10: 如何修改个人信息？
A: 点击顶部导航栏「我的」进入个人中心，可以修改密码。用户名暂不支持修改。
