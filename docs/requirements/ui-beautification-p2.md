## 1. 实施范围（P2 优先级）

### 1.1 能力雷达图（ECharts）
- 首页能力分布区域使用 ECharts 雷达图
- 显示 7 个安全分类的掌握情况
- 半透明渐变填充 + 交互提示

### 1.2 答题页全屏模式
- 添加全屏切换按钮
- 全屏时隐藏导航栏和非必要元素
- 专注模式：只保留题目、选项和倒计时

### 1.3 骨架屏加载状态
- 竞赛列表骨架屏
- 题目列表骨架屏
- 替代 Spinner，更专业的加载体验

### 1.4 竞赛卡片封面背景
- 不同分类对应不同渐变背景
- 抽象几何图案装饰

---

## 2. 技术约束

- 使用已安装的 ECharts + echarts-for-react
- 全屏使用浏览器 Fullscreen API
- 骨架屏使用 Tailwind + CSS 动画
- 不引入新的第三方库

---

## 3. 验收标准

- [ ] 首页雷达图正确显示 7 个分类数据
- [ ] 答题页有全屏切换按钮，全屏模式正常
- [ ] 竞赛列表有骨架屏加载状态
- [ ] 竞赛卡片有分类对应的渐变背景
- [ ] TypeScript 编译通过
- [ ] 移动端显示正常
