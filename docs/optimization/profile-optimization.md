# 个人中心界面优化方案

> **版本**: v1.0
> **创建日期**: 2026-04-03
> **状态**: 待确认

---

## 一、当前问题分析

| 模块 | 现状 | 问题 |
|------|------|------|
| **头像区域** | 默认图标，无用户头像 | 未显示用户真实头像 |
| **基本信息** | 用户名 + 注册时间 | 缺少等级、经验值、连续登录 |
| **统计数据** | 硬编码假数据 | 未从后端获取真实数据 |
| **布局** | 单卡片，简单排列 | 缺少 Bento 网格布局 |
| **视觉效果** | 无动画、无特效 | 缺少悬浮、交错入场等高级效果 |
| **功能** | 仅修改密码 | 缺少头像选择、目标管理 |

## 二、优化方案

### 2.1 整体布局（Bento Grid）

```
┌─────────────────────────────────────────────────────────┐
│  [👤 个人资料卡片 - 全宽]                                 │
│  [火影头像] 用户名  等级徽章  连续登录                    │
│  ████████████░░░░░░  经验值进度条                        │
│  [修改密码]  [选择头像]                                   │
├─────────────────────┬──────────────┬────────────────────┤
│  🏆 参赛次数 (1x1)  │  📊 总得分   │  🎯 当前排名 (1x1)  │
│                     │  (1x1)       │                    │
│        12           │     980      │     #15            │
│  ↑ 较上月 +3        │  ↑ +120      │  ↑ 较上周+2        │
├─────────────────────┴──────────────┴────────────────────┤
│  📈 各分类能力分布 (50%)         │  🎯 今日学习目标 (50%)  │
│  [雷达图]                        │  [进度条 x3]           │
├─────────────────────────────────┴───────────────────────┤
│  📋 最近参赛记录 (50%)           │  📊 答题统计 (50%)     │
│  [列表]                          │  [正确率/错题数]       │
└─────────────────────────────────────────────────────────┘
```

### 2.2 功能模块详情

#### 个人资料卡片
- **火影忍者头像**：从内置 21 个火影头像中选择，点击弹出头像选择器
- **用户名 + 等级徽章**：显示当前等级（学徒→大师）
- **连续登录天数**：🔥 连续 X 天
- **经验值进度条**：可视化显示升级进度
- **操作按钮**：修改密码 + 选择头像（弹窗选择内置头像）

#### 统计卡片（3 个）
- **参赛次数**：带趋势指示（↑/↓）
- **总得分**：带增长指示
- **当前排名**：带排名变化

#### 能力分布雷达图
- 7 个安全分类的正确率
- 与首页保持一致的 ECharts 雷达图

#### 今日学习目标
- 练习题进度
- 竞赛参与进度
- 错题复习进度

#### 最近参赛记录
- 最近 5 次参赛记录
- 显示竞赛名称、得分、排名、日期

#### 答题统计
- 总答题数
- 正确率
- 错题数
- 各分类答题分布

### 2.3 高级视觉效果（参考 VibeVibe 5.5）

| 效果 | 应用位置 | 说明 |
|------|---------|------|
| **悬浮抬起** | 所有卡片 | hover 上移 8px + 加深阴影 |
| **交错入场** | 所有卡片 | 依次入场，延迟 0.1s |
| **数字滚动** | 统计数字 | 从 0 增长到目标值 |
| **极光背景** | 页面背景 | 模糊渐变色块漂浮 |

## 三、API 需求

### 3.1 已有 API（可直接使用）

| 接口 | 用途 |
|------|------|
| `GET /api/user/profile` | 获取用户资料（含经验值、等级、连续登录、每日目标） |
| `GET /api/user/stats/trend` | 获取趋势数据（参赛次数、总得分、排名） |
| `GET /api/stats/category` | 获取各分类正确率 |
| `GET /api/stats/dashboard` | 获取最近提交记录 |
| `PUT /api/user/avatar` | 更换头像（选择内置头像路径） |

### 3.2 需要新增的 API

| 接口 | 用途 | 说明 |
|------|------|------|
| `GET /api/user/history` | 获取最近参赛记录 | 返回最近 N 次参赛详情 |
| `GET /api/user/answers` | 获取答题统计 | 总答题数、正确率、错题数、分类分布 |
| `GET /api/avatars` | 获取内置头像列表 | 返回所有可用头像路径（21 个火影头像） |

> **注意**：`PUT /api/user/avatar` 已存在，改为接收头像路径而非文件上传。

## 四、优先级排序

| 优先级 | 功能 | 工作量 | 效果 |
|--------|------|--------|------|
| **P0** | 个人资料卡片（头像+等级+经验值+连续登录） | 中 | 高 |
| **P0** | 统计卡片（真实数据+趋势指示） | 低 | 高 |
| **P0** | Bento 布局对齐 | 中 | 高 |
| **P1** | 雷达图 + 学习目标 | 低 | 中 |
| **P1** | 最近参赛记录 + 答题统计 | 中 | 中 |
| **P2** | 高级视觉效果（悬浮/交错/数字滚动/极光） | 低 | 中 |
