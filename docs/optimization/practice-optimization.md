# 题库练习界面优化方案

> **版本**: v1.0
> **创建日期**: 2026-04-03
> **状态**: 待确认

---

## 一、当前问题分析（优化前）

| 模块 | 现状 | 问题 |
|------|------|------|
| **筛选器** | 仅分类和难度两个下拉框 | 缺少题型筛选、搜索功能、标签筛选 |
| **题目列表** | 简单卡片列表，无分页 | 题目多时页面过长，无分页控制 |
| **题目卡片** | 信息展示不完整 | 缺少题型标签、正确率、练习次数 |
| **练习模式** | 单题练习，答完返回列表 | 缺少连续练习模式、答题进度、得分统计 |
| **布局** | 单列简单排列 | 缺少 Bento 网格布局、高级视觉效果 |
| **统计数据** | 无 | 缺少今日练习统计、正确率趋势 |

## 二、优化方案

### 2.1 整体布局（Bento Grid）

```
┌─────────────────────────────────────────────────────────┐
│  [📚 题库练习 - 页面标题 + 搜索框]                        │
├─────────────────────┬──────────────┬────────────────────┤
│  📝 今日练习 (1x1)  │  ✅ 正确率   │  🔥 连续练习 (1x1)  │
│                     │  (1x1)       │                    │
│        12           │     78%      │     5 天           │
│  ↑ 较昨日 +3        │  ↑ +5%       │  ↑ 较上周+2        │
├─────────────────────┴──────────────┴────────────────────┤
│  [🔍 高级筛选栏] 分类 | 难度 | 题型 | 搜索                │
├─────────────────────────────────────────────────────────┤
│  [题目列表 - 分页卡片]                                    │
│  ┌──────────────────────────────────────────────────┐   │
│  │ [简单] [单选] 以下哪种攻击方式属于注入攻击？        │   │
│  │ 分类: Web安全 | 2分 | 正确率: 78% | 已练习: 156次  │   │
│  │ [SQL注入] [注入攻击]              [开始练习 →]     │   │
│  └──────────────────────────────────────────────────┘   │
│  ┌──────────────────────────────────────────────────┐   │
│  │ ...                                              │   │
│  └──────────────────────────────────────────────────┘   │
│  [分页: < 1 2 3 4 5 >  共 46 题]                        │
└─────────────────────────────────────────────────────────┘
```

### 2.2 统计卡片（3 个）

| 卡片 | 内容 | 数据来源 |
|------|------|---------|
| **今日练习** | 今日答题数量 + 较昨日变化 | 后端统计 API |
| **正确率** | 今日答题正确率 + 趋势 | 后端统计 API |
| **连续练习** | 连续练习天数 | 后端统计 API |

### 2.3 高级筛选栏

| 筛选项 | 选项 | 说明 |
|--------|------|------|
| **分类** | 全部 / Web安全 / 密码学 / 逆向工程 / Pwn / Misc / 网络安全基础 / 操作系统安全 / 安全法规 | 下拉选择 |
| **难度** | 全部 / 简单 / 中等 / 困难 | 下拉选择 |
| **题型** | 全部 / 单选题 / 多选题 / 判断题 / 填空题 | 下拉选择 |
| **搜索** | 关键词搜索 | 输入框，支持题目内容搜索 |

### 2.4 题目卡片优化

**优化前**：
```
[简单] 题目内容（截断80字）
分类: Web安全 | 2分
[SQL注入] [注入攻击]    [开始练习]
```

**优化后**：
```
┌─────────────────────────────────────────────────────────┐
│  [简单] [单选题]                                         │
│  以下哪种攻击方式属于注入攻击？                           │
│                                                         │
│  📂 Web安全    💰 2分    ✅ 正确率 78%    📝 已练 156次  │
│  [SQL注入] [注入攻击]                    [开始练习 →]    │
└─────────────────────────────────────────────────────────┘
```

### 2.5 连续练习模式

**新增功能**：
- **顺序练习**：按题目列表顺序依次练习，自动切换下一题
- **随机练习**：随机抽取题目练习
- **错题重练**：只练习答错的题目
- **答题进度**：显示当前进度（第 X/Y 题）
- **得分统计**：练习结束后显示本次得分

### 2.6 高级视觉效果（参考 VibeVibe 5.5）

| 效果 | 应用位置 | 说明 |
|------|---------|------|
| **悬浮抬起** | 所有卡片 | hover 上移 8px + 加深阴影 |
| **交错入场** | 题目卡片 | 依次入场，延迟 0.05s |
| **数字滚动** | 统计数字 | 从 0 增长到目标值 |
| **极光背景** | 页面背景 | 模糊渐变色块漂浮 |
| **骨架屏** | 加载状态 | 替代 Spinner |

## 三、API 需求

### 3.1 已有 API（可直接使用）

| 接口 | 用途 |
|------|------|
| `GET /api/practice` | 获取练习题目列表（支持分页、筛选） |
| `POST /api/practice/:questionId/answer` | 提交练习答案 |
| `GET /api/user/answers` | 获取答题统计 |

### 3.2 需要新增的 API

| 接口 | 用途 | 说明 |
|------|------|------|
| `GET /api/user/practice-stats` | 获取练习统计（今日练习、正确率、连续天数） | 返回今日练习数、正确率、连续练习天数 |

## 四、优先级排序

| 优先级 | 功能 | 工作量 | 效果 |
|--------|------|--------|------|
| **P0** | 统计卡片（今日练习、正确率、连续练习） | 中 | 高 |
| **P0** | 题目卡片优化（题型标签、正确率、练习次数） | 中 | 高 |
| **P0** | 分页功能 | 低 | 高 |
| **P1** | 高级筛选栏（题型筛选 + 搜索） | 低 | 中 |
| **P1** | 连续练习模式（顺序/随机/错题） | 高 | 中 |
| **P2** | 高级视觉效果（悬浮/交错/数字滚动/极光） | 低 | 中 |
| **P2** | 骨架屏加载状态 | 低 | 低 |
