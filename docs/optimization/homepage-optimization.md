## 三、待优化功能

### 3.1 Bento 布局对齐优化

**当前问题**：
- 卡片高度不一致，导致网格出现大片空白
- 雷达图卡片和下一场比赛卡片高度不匹配
- 移动端和桌面端布局需要更精细的响应式控制

**优化方案**：

```
┌─────────────────────────────────────────────────────────┐
│  [欢迎区 - 全宽]                                         │
│  [👤图片头像] 👋 早上好，张三                             │
│  🔥 连续登录 7 天    🏅 安全工程师 Lv.4                  │
│  ████████████░░░░░░  67/100 经验值                       │
│  [⚡ 快速练习]  [🏁 参加竞赛]  [📊 查看排名]  [📝 错题复习] │
├─────────────────────┬──────────────┬────────────────────┤
│  🏁 正在进行 (2x1)  │  📊 平均得分  │  🏆 当前排名 (1x1)  │
│                     │  (1x1)       │                    │
│        3            │     78.5     │     #15            │
│  ↑ 较昨日 +1        │  ↑ +2.3%     │  ↑ 较上周+3        │
├─────────────────────┴──────────────┴────────────────────┤
│  📈 能力分布 (50%)              │  🎯 今日学习目标 (50%)  │
│  [雷达图]                       │  [进度条 x3]           │
├─────────────────────────────────┴───────────────────────┤
│  💡 每日安全小贴士 (50%)         │  📋 最近提交 (50%)     │
│  [切换提示]                      │  [列表]               │
└─────────────────────────────────────────────────────────┘
```

**布局规则**：
- 使用 CSS Grid 替代 Ant Design Row/Col，更精确控制对齐
- 所有卡片设置 `min-h` 确保高度一致
- 雷达图和学习目标同行，高度统一为 280px
- 安全小贴士和最近提交同行，高度统一为 200px
- 移动端自动堆叠为单列

### 3.2 高级视觉效果（参考 VibeVibe 5.5 节）

**使用原则**：
- 一个页面最多用 2-3 个高级效果
- 先做功能，再加效果
- 移动端谨慎使用复杂动画
- 每个动画都应该传达信息或引导注意力

#### 效果 1：悬浮抬起（Hover Lift）— 低难度
- **应用位置**：所有 Bento 卡片
- **效果描述**：hover 时卡片向上移动 8px 并加深阴影
- **实现方式**：纯 CSS transition
- **AI 提示词**：给卡片加悬浮抬起效果，hover 时向上移动 8px 并加深阴影，用 CSS transition

#### 效果 2：交错入场（Stagger Animation）— 低难度
- **应用位置**：首页所有卡片
- **效果描述**：卡片依次交错入场，每个延迟 0.1 秒，从透明+缩小到正常状态
- **实现方式**：Framer Motion staggerChildren
- **AI 提示词**：让网格中的卡片依次交错入场，每个延迟 0.1 秒，从透明+缩小到正常状态

#### 效果 3：数字滚动（Counter Animation）— 中难度
- **应用位置**：统计卡片中的数字
- **效果描述**：数字从 0 增长到目标值，滚动到视口时触发，持续 1.5 秒
- **实现方式**：Framer Motion useTransform + useScroll
- **AI 提示词**：做一个数字滚动动画组件，数字从 0 增长到目标值，滚动到视口时触发，持续 1.5 秒

#### 效果 4：极光背景（Aurora Background）— 低难度
- **应用位置**：首页背景装饰
- **效果描述**：多个模糊渐变色块独立漂浮移动，营造北极光氛围
- **实现方式**：CSS filter blur + animation
- **AI 提示词**：给首页加极光背景效果，用多个模糊渐变色块独立漂浮移动，营造北极光氛围，用 CSS filter blur + animation 实现

#### 效果 5：发光脉冲按钮（Glow Pulse Button）— 低难度
- **应用位置**："立即参赛"等 CTA 按钮
- **效果描述**：持续呼吸式发光，hover 时光晕增强变色
- **实现方式**：box-shadow + animation
- **AI 提示词**：做一个发光脉冲按钮，持续呼吸式发光，hover 时光晕增强变色，用 box-shadow + animation 实现

#### 效果 6：滚动触发动画（Scroll-triggered Animation）— 低难度
- **应用位置**：首页所有模块
- **效果描述**：每个模块在滚动到视口时从下方淡入
- **实现方式**：Framer Motion whileInView
- **AI 提示词**：让每个功能卡片在滚动到视口时从下方淡入，用 Motion 的 whileInView，每个卡片延迟 0.1 秒
