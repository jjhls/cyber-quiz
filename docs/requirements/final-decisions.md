# 网络安全竞赛平台 - 最终决策汇总

> 基于两轮访谈确认 + UI 方案讨论，以下为最终冻结的决策清单。
> 如有异议请在进入执行计划前提出。

---

## 一、功能决策

| 维度 | 决策 |
|------|------|
| **用户角色** | 参赛者 + 超级管理员（兼出题和管理） |
| **题目类型** | 单选题、多选题、判断题、填空题（全部自动判分） |
| **题目分类** | Web安全、密码学、逆向工程、Pwn、Misc、网络安全基础、操作系统安全、安全法规与合规 |
| **题目属性** | 难度等级、分值、标签/关键词、题目解析 |
| **竞赛模式** | 限时竞赛 + 自由练习 |
| **比赛形式** | 个人赛，自由报名 |
| **计分规则** | 答对得分，答错不扣分 |
| **排名规则** | 按总分排名，同分按用时排序 |
| **防作弊** | 题目随机顺序 + 选项随机顺序 + 限时答题 |
| **排行榜** | 实时排行榜 |
| **成绩功能** | 历史成绩记录 + 错题本 + 数据统计面板 |
| **移动端** | 响应式设计 + 移动端底部 Tab 导航 |
| **规模** | 100人以内，并发 < 50 |

## 二、技术决策

| 维度 | 决策 | 理由 |
|------|------|------|
| **前端框架** | React 18 + TypeScript | 类型安全，生态成熟 |
| **UI 组件库** | Ant Design 5 | 组件丰富，主题可定制 |
| **样式方案** | Tailwind CSS | 原子化 CSS，Bento 布局友好 |
| **配色方案** | 暗色模式，slate 背景 + blue 主色 | 专业克制，现代感 |
| **布局风格** | Bento Grid 网格布局 | 模块化，秩序感 |
| **图表库** | ECharts | 雷达图、柱状图完善 |
| **动画库** | Framer Motion | React 动画首选 |
| **图标库** | Lucide React | 现代简洁 |
| **路由** | React Router v6 | 标准选择 |
| **状态管理** | Zustand | 轻量简洁 |
| **表单** | React Hook Form + Zod | 类型安全验证 |
| **后端框架** | Express（默认） | 轻量，适合 MVP 规模 |
| **认证方式** | Session + Cookie | 实现简单，适合单体/同域部署，无需处理 token 刷新 |
| **数据库** | SQLite + Prisma ORM | 轻量开发，预留迁移能力 |
| **项目结构** | Monorepo（默认） | 前后端同仓，方便管理 |
| **题库导入** | CSV 批量导入（默认） | 快速建题库 |
| **竞赛时间** | 固定时间段（默认） | 实现简单 |
| **答题交互** | 不可暂停 + 支持标记 + 自由切换（默认） | 标准竞赛体验 |
| **填空题判分** | 忽略大小写和首尾空格 + 支持多答案（默认） | 兼顾严格与容错 |
| **多选题判分** | 全对才得分（默认） | 规则简单明确 |
