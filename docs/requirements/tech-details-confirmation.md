## 1. 后端框架选择

Node.js 生态有两个主流选项：

- [x] **Express**：轻量灵活，上手快，适合小型项目

## 2. 前端移动端适配方式

你提到了移动端适配，有两种实现方式：

- [x] **响应式设计**：一套代码，通过 CSS 媒体查询自适应手机/平板/PC（推荐 MVP）
- [ ] **PWA（渐进式 Web 应用）**：响应式 + 可添加到手机桌面 + 离线缓存
- [ ] **独立移动端页面**：单独做一套手机端页面

> 💡 建议 MVP 用响应式设计，成本低、维护简单。

## 3. 用户认证方式

- [ ] **JWT（JSON Web Token）**：无状态，适合前后端分离，token 过期需刷新
- [x] **Session + Cookie**：有状态，服务端管理会话，实现简单

> 💡 前后端分离推荐 JWT；如果想简单快速，Session 也可以。
