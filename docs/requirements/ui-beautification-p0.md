## 1. 实施范围（P0 优先级）

### 1.1 背景网格/渐变
- 页面背景添加微妙的蓝色网格图案（opacity 极低）
- 顶部添加径向渐变聚光灯效果
- 不影响现有内容可读性

### 1.2 卡片边框优化
- 卡片 hover 时边框颜色从 slate-700 变为 blue-500/30
- 卡片添加 hover 时顶部蓝色高亮条（2px）
- 保持现有 rounded-2xl 圆角

### 1.3 答题结果页分数动画
- 分数使用数字滚动动画（从 0 滚动到实际分数）
- 添加圆形进度图（SVG）展示得分率
- 根据分数显示不同评价文字和表情

### 1.4 倒计时圆形进度环
- 答题页倒计时使用 SVG 圆形进度环
- 进度环颜色随剩余时间变化（蓝→橙→红）
- 最后 5 分钟添加脉冲边框动画

---

## 2. 技术约束

- 不引入新的第三方库（使用已有 Tailwind + Framer Motion）
- 不改变现有布局结构
- 保持暗色主题一致性
- 移动端适配不受影响

---

## 3. 验收标准

- [ ] 页面背景有网格/渐变效果
- [ ] 卡片 hover 有顶部高亮条 + 边框变色
- [ ] 答题结果页分数有滚动动画 + 圆形进度图 + 评价
- [ ] 答题页倒计时有圆形进度环 + 颜色变化
- [ ] 所有改动 TypeScript 编译通过
- [ ] 移动端显示正常
